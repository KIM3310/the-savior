서비스 운영 모델

The Savior는 별도 유료 구독 없이 운영됩니다. AI 기능은 사용자 OpenAI API 키(BYOK)를 우선 사용하고, 서비스 수익은 Google AdSense 광고를 기반으로 합니다.

1. AI 사용 방식

  • 사용자가 메인 화면에서 자신의 OpenAI API 키를 입력해 사용합니다.
  • 개인 키가 없을 때만 서버 기본 키를 폴백으로 사용합니다.
  • 키는 사용자 기기 로컬 저장소에 저장되며, 요청 시에만 전송됩니다.

2. 수익 방식

  • Google AdSense 광고 슬롯 기반 수익화
  • 쿠키 동의 시 광고 및 성능 측정 기능 활성화
  • 광고 관련 정책 문서(약관/개인정보처리방침) 상시 제공

3. 서비스 원칙

본 서비스는 웰니스 목적의 도구이며 의료 행위를 대체하지 않습니다. 위기 신호가 감지되면 긴급기관 및 전문기관 안내를 우선 제공합니다.
